Política de Privacidade

Última atualização: 1 de outubro de 2025

Esta Política descreve como a Extensão Concursos Públicos coleta, usa e protege informações quando você a instala e utiliza no navegador. A extensão foi desenvolvida para ajudar candidatos a concursos públicos a organizar editais, cronogramas e materiais de estudo diretamente no navegador.

1. Coleta de Dados

A extensão não coleta dados pessoais identificáveis de forma remota. Todas as informações que a extensão manipula — como favoritos, lembretes e preferências — ficam armazenadas localmente no navegador do usuário via API de armazenamento do Chrome (chrome.storage).

2. Uso de Permissões

A extensão solicita permissões mínimas estritamente necessárias para seu funcionamento. Abaixo há uma explicação das permissões utilizadas e suas finalidades:

  • Permissão de host (acesso a páginas web): usada apenas para ler conteúdo público de sites de concursos quando o usuário solicita a importação/visualização de um edital ou atualizar dados de um concurso. A extensão não altera conteúdos nem envia informações desses sites para servidores externos.
  • storage: usado para salvar configurações e preferências do usuário, como concursos salvos, lembretes e filtros. O armazenamento é local ao navegador e não é sincronizado automaticamente com servidores externos sem o consentimento explícito do usuário.
  • Código remoto (se aplicável): caso a extensão carregue bibliotecas externas (CDNs) ou faça requisições a APIs públicas para obter dados de editais, isso será utilizado apenas para oferecer atualizações em tempo real e funcionalidades essenciais. Nenhum código remoto é usado para coletar dados pessoais sem consentimento.

3. Uso dos Dados

Os dados armazenados localmente são utilizados exclusivamente para proporcionar a funcionalidade da extensão: organizar editais, criar lembretes de provas, destacar trechos e salvar preferências. Não vendemos, alugamos ou compartilhamos dados pessoais com terceiros.

4. Compartilhamento de Dados

Não compartilhamos informações pessoais com terceiros, salvo quando exigido por lei ou mediante ordem judicial. Se em algum momento for necessária a integração com serviços externos (por exemplo, sincronização na nuvem), isso ocorrerá apenas com o consentimento explícito do usuário e será informado com antecedência.

5. Segurança

As informações permanecem no dispositivo do usuário e são protegidas pelos mecanismos de segurança do navegador. Recomendamos que os usuários mantenham o navegador atualizado para garantir a melhor proteção possível.

6. Alterações nesta Política

Podemos atualizar esta Política periodicamente para refletir mudanças nas práticas do produto ou exigências legais. Quando houver alterações significativas, publicaremos a data da última atualização nesta página.
